My personal ESLint config

## Features

- Support ESM module
- Compatible with Prettier

## Install

```bash
npm i -D eslint @fzf404/eslint-config
```

## Usage

Edit `package.json` or `.eslintrc.json`

```jsonc
// package.json
"eslintConfig": {
		"extends": [
			"@fzf404/eslint-config"
		]
}

// .eslintrc.json
{
  "extends": [
    "@fzf404/eslint-config"
  ]
}
```
